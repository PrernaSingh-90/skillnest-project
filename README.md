# SkillNest Pro – MERN Stack Application

This is a full-stack MERN (MongoDB, Express, React, Node.js) application built as part of an assignment. The project is production-ready with authentication, protected routes, and CRUD functionality.

---

## 🚀 Features

- User Authentication (JWT-based login/signup)
- Protected Routes
- Course Management (CRUD operations)
- Student Enrollment System
- Responsive UI
- REST API integration

---

## 🛠 Tech Stack

**Frontend:**
- React.js
- CSS 

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## ▶️ Run Project

### Backend

```
cd server
npm install
npm start
```

### Frontend

```
cd client
npm install
npm start
```

---
